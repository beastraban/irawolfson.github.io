SunBURST: GPU-Accelerated Bayesian Evidence Calculation
I. Wolfson — In preparation for MNRAS

Bayesian model comparison requires computing intractable high-dimensional integrals. SunBURST exploits GPU parallelism to achieve O(d0.67) scaling through 1024+ dimensions — a paradigm shift enabling model selection problems previously considered computationally impossible. The framework represents a breakthrough in GPU-accelerated Bayesian evidence calculation.

See the dedicated SunBURST project page for more details.
