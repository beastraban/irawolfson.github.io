SunBURST achieves O(d0.67) scaling through 512+ dimensions—a breakthrough in computational Bayesian inference that enables model comparison problems previously considered intractable.

What It Does

Bayesian model comparison requires computing the evidence (marginal likelihood) for competing models. This integral over parameter space is notoriously difficult to calculate in high dimensions. SunBURST exploits GPU parallelism to make this tractable for real-world problems.

Key Features

  • Dimensional scaling: O(d0.67) tested through 512 dimensions
  • GPU acceleration: CUDA-based parallel computation
  • Local execution: Runs on your hardware, using your GPUs
  • Python interface: Clean API for integration with existing workflows
  • Machine precision: Validated against analytical solutions for Gaussian posteriors

Performance

Benchmarked against established nested sampling tools (dynesty, ultranest, nautilus), SunBURST demonstrates 245× speedup at 32 dimensions for Gaussian posteriors while maintaining machine precision accuracy.

Architecture

SunBURST runs locally on your machine. The interface connects to a local server, keeping all computation on your infrastructure:

  1. Install SunBURST package (Python + CUDA)
  2. Launch local server
  3. Define models, run evidence calculations, visualize results

This architecture means zero cloud costs and complete control over your data.
